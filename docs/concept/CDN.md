`CDN网络是在用户和服务器之间增加Cache层，如何将用户的请求引导到Cache上获得源服务器的数据`

用户访问未使用 CDN 缓存网站的过程为:

- 用户向浏览器提供要访问的域名；
- 浏览器调用域名解析函数库对域名进行解析，以得到此域名对应的 IP 地址；
- 浏览器使用所得到的 IP 地址，域名的服务主机发出数据访问请求；
- 浏览器根据域名主机返回的数据显示网页的内容。
