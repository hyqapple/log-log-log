# 经典布局

## 居中布局

### 水平居中

#### inline-block text-align

先将子框由块级元素改变为行内块元素，再通过设置行内块元素居中以达到水平居中。

```html
<div class="parent">
  <div class="child">DEMO</div>
</div>
```

```css
.child {
  display: inline-block;
}
.parent {
  text-align: center;
}
```
